Developed ZiPack, a high-fidelity, real-time AI-powered personal nutrition and meal planning assistant, showcasing advanced full-stack application development. The frontend, built with React, provides intuitive nutrient tracking progress displays, dynamic weight history charting, and a user-friendly interface enhanced by subtle micro-interactions for an elevated user experience. All visual elements are meticulously crafted and styled with a sophisticated dark theme, drawing inspiration from professional development environments. The secure backend leverages Firebase, utilizing Firestore for real-time user profile, meal log, pantry, and recipe data synchronization, and Firebase Authentication for robust anonymous and secure user management. Key challenges overcome include ensuring instantaneous real-time data synchronization across all connected clients, implementing secure user authentication flows that prioritize data privacy, and seamlessly integrating with the Gemini API to deliver personalized, intelligent coaching insights and on-demand recipe generation based on user profiles and pantry inventory. Furthermore, the application features a comprehensive meal logging system for daily dietary tracking, a smart pantry management function that highlights expiring items to minimize food waste, and a global toast notification system for immediate user feedback, all contributing to a truly engaging and productive personal health and culinary journey.
