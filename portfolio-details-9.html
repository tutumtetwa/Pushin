The Vision: An AI-Powered Nutrition Assistant

ZiPack is a high-fidelity, real-time AI-powered personal nutrition and meal planning assistant, built to showcase advanced full-stack application development. The application is designed to guide users on an engaging and productive personal health and culinary journey through intelligent, data-driven insights.

Full-Stack Architecture

  • Frontend: React, Custom Charting & UI
  • Backend: Firebase (Firestore, Authentication)
  • AI Integration: Google Gemini API
  • Key Concept: Real-time Data Synchronization

Core Feature: The Gemini AI Engine

The core of the application is its seamless integration with the Google Gemini API. This delivers personalized coaching insights and enables on-demand recipe generation based on the user's profile and live pantry inventory. A key feature is the smart pantry management system, which highlights expiring items to minimize food waste and suggest recipes accordingly.

Intuitive Health & Pantry Management

The React frontend provides an intuitive user experience with nutrient tracking displays, dynamic weight history charting, and subtle micro-interactions within a sophisticated dark theme. Real-time data synchronization with Firestore and a global toast notification system ensure users always have immediate feedback and up-to-date information for their meal logging and dietary tracking.
