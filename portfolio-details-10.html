ZiPack(iOS App)

Project information

  • Category: App
  • Client: ZiPack
  • Project date: July, 2025
  • Github: Website

I created ZiPack, an intelligent native iOS application developed with Xcode and SwiftUI, designed to be your personal chef and nutrition companion. It's built on Firebase for robust data management, using Firestore to securely store user profiles, pantry items, and meal logs, and Firebase Authentication for streamlined anonymous access. The app's personalization magic comes from Google Gemini, an advanced AI model that dynamically generates tailored recipes and daily meal plans, considering everything from your profile to expiring pantry items. Key integrations include the Firebase iOS SDK, GoogleGenerativeAI, and a Charts Library for clear weight progress visualization.
