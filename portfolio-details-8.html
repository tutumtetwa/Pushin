Architected and implemented Planna, a high-fidelity, real-time collaborative Kanban board, demonstrating expertise in complex full-stack application development. The frontend, built with React, features a custom drag-and-drop system that allows intuitive task and column reordering, dynamic filtering and sorting capabilities for enhanced task management, and an intuitive UI with subtle micro-interactions that elevate the user experience. All visual elements are meticulously crafted and styled with a sophisticated dark theme, drawing inspiration from professional development environments. The secure backend, powered by Node.js/Express, utilizes Firebase Admin SDK for privileged interactions with Firestore and Firebase Authentication, handling robust user registration with bcrypt for secure PIN hashing, streamlined login processes, and reliable account recovery. Key challenges addressed include ensuring instantaneous real-time data synchronization across all connected clients, implementing secure custom authentication flows that prioritize user data protection, and optimizing frontend performance for dynamic content manipulation and complex filtering operations, resulting in a highly responsive, scalable, and robust project management tool. Furthermore, the application incorporates a comprehensive activity log for transparency, a flexible task archiving system for efficient board decluttering, and a global toast notification system for immediate user feedback, contributing to a truly engaging and productive collaborative workspace.
