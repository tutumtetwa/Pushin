As an active trader seeking to move beyond scattered notes and gain a data-driven edge, I developed BreadBook to solve a personal need for a centralized, all-in-one tool. This project was conceived as a practical solution to organize my daily workflow and has since become an indispensable part of my trading discipline. The front-end is a dynamic single-page application built with React, featuring a responsive and intuitive UI styled with Tailwind CSS. To create a polished and engaging user experience, the interface is enhanced with dynamic animations, including a shimmering gradient title and a personalized header. A key feature is the interactive dashboard, which uses the Recharts library to generate a real-time equity curve and a monthly calendar heatmap, allowing for at-a-glance performance review. The entire backend is serverless, leveraging a full suite of Firebase services. Firestore provides a real-time NoSQL database that instantly syncs all CRUD (Create, Read, Update, Delete) operations on trades across sessions. Firebase Authentication handles seamless and secure anonymous user accounts, while Firebase Storage is used to host chart screenshots uploaded by the user for later review. The entire application is deployed and hosted globally via Firebase Hosting. Using BreadBook daily has been transformative. It has centralized my record-keeping, helped me identify winning and losing patterns through clear data visualization, and provided the critical, organized feedback loop needed to consistently improve my strategies.
