Project Overview

As an active trader seeking to move beyond scattered notes and gain a data-driven edge, I developed BreadBook to solve a personal need for a centralized, all-in-one tool. This project was conceived as a practical solution to organize my daily workflow and has since become an indispensable part of my trading discipline.

Tech Stack & Features

  • Frontend: React, Tailwind CSS, Recharts
  • Backend: Firebase (Firestore, Authentication, Storage, Hosting)
  • Key Features: Real-time data sync, CRUD operations, interactive visualizations (equity curve, calendar), goal tracking, and image uploads.

The front-end is a dynamic single-page application built with React, featuring a responsive UI styled with Tailwind CSS and enhanced with dynamic animations. The interactive dashboard uses the Recharts library to generate a real-time equity curve and a monthly calendar heatmap for at-a-glance performance review. The entire backend is serverless, leveraging a full suite of Firebase services to handle the database, user authentication, and file storage.

Project Impact

Using BreadBook daily has been transformative. It has centralized my record-keeping, helped me identify winning and losing patterns through clear data visualization, and provided the critical, organized feedback loop needed to consistently improve my strategies.
