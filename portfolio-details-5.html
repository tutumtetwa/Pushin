SmartNotes

Project Information

  • Category: Web App
  • Name: SmartNotes
  • Project Date: May, 2025
  • Project URL: Website
  • Project Github Repository: Github

Developed a full-stack AI-powered note-taking web application that allows users to create, store, and manage notes enriched with advanced NLP capabilities. Integrated Hugging Face Transformers to enable automatic summarization and semantic search, allowing users to extract key insights and retrieve relevant notes using natural language queries. Built using Flask for the backend, PostgreSQL for persistent storage, and HTML/CSS/JS for the frontend interface. The system processes user input through an embedding pipeline to match contextually relevant content, enhancing productivity and information retrieval efficiency.
